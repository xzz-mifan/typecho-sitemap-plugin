Sitemap 是一款为 Typecho 编写的站点地图插件，有助于网站的 SEO 和收录。

## 安装方法

1. 前往[Releases](https://github.com/xzz-mifan/typecho-sitemap-plugin/releases) 下载最新版本插件
2. 解压后重命名文件夹为 `Sitemap` ，再上传至网站的 `/usr/plugins/` 目录下
3. 启用该插件即可，无需任何设置
